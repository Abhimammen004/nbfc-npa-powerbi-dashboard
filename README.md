# NBFC NPA Portfolio Tracker

Interactive Power BI dashboard tracking NPA trends across a synthetic ₹7,600 Cr NBFC loan portfolio with RBI-calibrated data.

## Pages
- Executive Summary — KPI cards, donut chart, NPA rate by loan type
- Trend Analysis — GNPA vs NNPA trend, Provision Coverage Ratio (FY22–FY25)
- Branch Analysis — Branch-level NPA exposure and asset classification
- Portfolio Detail — Loan-level drill down with slicers and scatter chart
- Risk Metrics — Gauge, KPI visual, NPA vs Provisions by loan type

## Tools Used
Power BI, DAX, Python (synthetic data generation), RBI-calibrated NBFC data

## Key Metrics
- Portfolio Size: ₹7,600 Cr
- Gross NPA Rate: ~4.8%
- Asset Classes: Substandard, Doubtful, Loss
- Branches: 8 cities across India
